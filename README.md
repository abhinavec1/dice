This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Search the results using the Input box given on the page. (Press Enter to execute the search)

Use the Sort Button to sort the cards based on the fields provided. (The sorting has been implemented on the Frontend)
